# google_AIML

This repository was initialized by GitHub Copilot to allow creating the branch `save-from-colab` for saving work from Google Colab.

Branch purpose: store notebooks and files saved from Google Colab.

How to use:
- In Colab: File -> Save a copy in GitHub, choose branch `save-from-colab`.
- Or push notebooks to this branch using git from a local clone.
